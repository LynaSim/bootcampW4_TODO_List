# Bootcamp Week 4 assignment

## Create an interactive TODO list application.

I created a mini form that lets the user:<br/><br/>
* input the name of the task<br/>
* add the task to a list<br/>
* check a box to mark the task as done, triggering a fun confetti animation (confetti explosion)<br/>
* delete a task at the click of a button<br/>
* edit the task and save edits by entering an "edit mode" state.<br/>

Additionally the form will:<br/><br/>
* block an empty entry<br/>
* prompt the user to type a value, by way of an alert box<br/>
* allow submission by either clicking button or pressing the key "enter"
